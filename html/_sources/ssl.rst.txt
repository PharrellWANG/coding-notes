Definition
----------
Transport Layer Security (TLS) and its predecessor,
Secure Sockets Layer (SSL), are cryptographic protocols
that provide communications security over a computer network.
(1. `Transport Layer Security (TLS) <https://en.wikipedia.org/wiki/Transport_Layer_Security>`_)
