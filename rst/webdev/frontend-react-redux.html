How to escape single quote in HTML¶

You can also use &apos;, which is iffy in IE, or &#39;, which should work everywhere.

That is to say:

  • &#39; or &apos; for '

  • &#34; for ''

Reference: How do I escape a single quote? For more: Character entity references in HTML
