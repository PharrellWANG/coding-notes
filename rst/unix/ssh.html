6. SSH¶

6.1. Graphical Forwarding¶

How do I run graphical programs remotely from a Linux server?

To use matlab from v100t htgc code testing node on macos:

  1. open xQuartz

  2. ssh -Y htgc-code-testing-node

  3. matlab

Done.
