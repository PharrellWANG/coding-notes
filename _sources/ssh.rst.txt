===

Graphical Forwarding
--------------------

`How do I run graphical programs remotely from a Linux server? <https://uisapp2.iu.edu/confluence-prd/pages/viewpage.action?pageId=280461906>`_


To use matlab from v100t htgc code testing node on macos:

1. open xQuartz
2. ``ssh -Y htgc-code-testing-node``
3. ``matlab``

Done.
