8. Ports¶

8.1. Check what ports are listening on linux(ubuntu)¶

$ apt-get install nmap
$ nmap -sT -0 localhost

8.2. How to check opened/closed port on my computer?¶

https://askubuntu.com/questions/538208/how-to-check-opened-closed-port-on-my-computer

8.3. Kill ports¶

8.3.1. Kill ports on macOS¶

Tip

Automated script: see Freeport in GitHub. Sometimes it does not work fine.

Step 1 Find out the process ID (PID) which is occupying the port number (e.g., 5955) you would like to free

$ sudo lsof -t -i :3333

Step 2 Kill the process which is currently using the port using its PID

$ sudo kill -9 `PID`

E.g., to kill a process with PID 1234

$ sudo kill -9 1243

8.3.2. Kill ports on Linux¶

E.g., to free port 8081:

$ sudo kill `sudo lsof -t -i:8081`

Or, you can use (some times this is much more useful than the previous one):

$ fuser -n tcp -k 8080

8.4. What is special about port 6000 on macOS¶

Browsers block certain ports, although they are not in the system port range, some of them in the ranges widely used for local web development.
